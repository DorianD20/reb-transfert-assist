Ajout d'un téléphone en Bluetooth

Étape 1

Allez dans le menu, recherchez "Bluetooth" puis validez "ok".

Étape 2

Sur "Éditer", passez sur "on" puis validez "ok".

Étape 3

Sur "Appairer", validez "ok".

Étape 4

Le module doit afficher son id Bluetooth.

Étape 5

Recherchez dans les réglages Bluetooth du téléphone le code suivant pour appairer le module MASTER, puis jumelez.

Étape 6

Une fois la connexion établie, le sigle apparaît en haut de l'écran.
